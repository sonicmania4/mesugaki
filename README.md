# メスガキ変換機 ♡ ざぁこ♡ざぁこ♡

どんな文章も煽りモードに変換しちゃうメスガキ変換アプリです！

## 🎮 デモ

https://sonicmania4.github.io/mesugaki/

## ✨ 機能

- **3段階の煽りレベル**
  - マイルド（優しめの煽り）😊
  - スタンダード（定番メスガキ）😏
  - 激辛（超煽りモード）😈

- **AI変換機能**
  - Claude AIが自然なメスガキ口調に変換
  - 元の意味を保ちながら煽り要素を追加
  - 「ざぁこ♡」「よわよわ〜」などの定番ワードを使用

## 🔑 使い方

1. [Anthropic Console](https://console.anthropic.com/)でAPIキーを取得
2. アプリにAPIキーを入力
3. 変換したい文章を入力
4. メスガキレベルを選択
5. 「変換する ♡」をクリック！

## 💡 変換例

```
元: おはようございます
後: おっはよ〜♡ まだ寝てたのぉ？ざぁこ♡

元: お疲れ様でした
後: あはは♡ もう疲れちゃったの〜？よわよわだね〜💕

元: ありがとうございます
後: えへへ〜♡ 感謝しちゃう？かわいい〜💗
```
